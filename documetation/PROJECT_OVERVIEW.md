# Project Overview - Pradeep Kumar S Portfolio

A modern, high-performance portfolio website built with React 19, Vite, and Tailwind CSS v4. This project showcases technical skills, experience, and projects with a focus on speed, accessibility, and a premium "dashboard-like" user interface.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) (Functional Components, Hooks)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)

## Key features

- **Dynamic Interactive UI**: Real-time tab Switching and smooth transitions between sections.
- **Micro-Animations**: Uses Framer Motion for high-end UI effects like the "Portal" zoom sequence.
- **Advanced SEO**: Pre-configured with Open Graph meta tags, Twitter cards, and JSON-LD structured data for better search engine visibility.
- **Accessibility (A11y)**: Semantic HTML structure, high contrast ratios, and keyboard-friendly navigation.
- **Responsive Design**: Optimized for everything from mobile devices to ultra-wide monitors.
- **Custom SSG Pipeline**: A specialized build process that ensures high performance and SEO benefits.
